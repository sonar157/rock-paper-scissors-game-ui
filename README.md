# Rock Paper Scissors Game ✊📄✂

A simple interactive Rock Paper Scissors game built using HTML, CSS, and JavaScript.  
The user selects rock, paper, or scissors and the computer randomly generates its choice.  
The result is displayed instantly.

## Features 🚀
- Interactive buttons with hover and click effects
- Random computer choice generation
- Instant result display (Win / Lose / Draw)
- Simple and clean UI
- Beginner friendly JavaScript logic

## Technologies Used 💻
- HTML
- CSS
- JavaScript

## How It Works ⚙️
1. User clicks on Rock, Paper, or Scissors button.
2. Computer randomly selects one option.
3. The program compares both choices.
4. Result is displayed on the screen.

## Game Rules 📜
- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock
- Same choice results in Draw
